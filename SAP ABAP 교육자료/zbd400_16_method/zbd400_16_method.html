*&---------------------------------------------------------------------*
*& Report ZBD400_16_METHOD
*&---------------------------------------------------------------------*
*&
*&---------------------------------------------------------------------*
REPORT ZBD400_16_METHOD.


PARAMETERS P_BASE TYPE I.
PARAMETERS P_POWER TYPE I.


START-OF-SELECTION.


DATA LV_BASE TYPE BC400_COMPUTE_BASE.
DATA LV_POWER TYPE BC400_COMPUTE_POWER.
DATA LV_RESULT TYPE BC400_COMPUTE_RESULT.

LV_BASE = P_BASE.
LV_POWER = P_POWER.

* TRY를 사용해야 CATCH를 사용할 수 있음.

TRY.
CALL METHOD CL_BC400_COMPUTE=>GET_POWER
  EXPORTING
    IV_BASE   =    LV_BASE               " Base Value for Power Calculation
    IV_POWER  =    LV_POWER              " Exponent for Power Calculation
  IMPORTING
    EV_RESULT =    LV_RESULT              " Result of Calculation with Two Decimal Places
  .

WRITE LV_RESULT. " EXCEPTION이 발생하지 않으면 실행.

* EXCEPTION이 가동하면 PROGRAMME이 DUMP 되지만 CATCH를 통해 DUMP를 막음.

 CATCH CX_BC400_POWER_TOO_HIGH.  " The maximum value for the exponent is 4 " GET_POWER에 있는 EXCEPTION.

   WRITE 'MAX VALUE FOR POWER IS 4.'.

 CATCH CX_BC400_RESULT_TOO_HIGH. " Result of calculation too high for result variable " 이것도 EXCEPTION.

   WRITE 'RESULT VALUE WAS TOO HIGH.'.

ENDTRY.
