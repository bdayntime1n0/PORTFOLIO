Description: SELCETION SCREEN TABSTRIP 예제
*&---------------------------------------------------------------------*
*& Report ZBD430_16_SSCR_TABSTRIP
*&---------------------------------------------------------------------*
*&
*&---------------------------------------------------------------------*
REPORT ZBD430_16_SSCR_TABSTRIP.

SELECTION-SCREEN BEGIN OF SCREEN 1100 AS SUBSCREEN.
  PARAMETERS : P_CARR1 TYPE SCARR-CARRID.
SELECTION-SCREEN END OF SCREEN 1100.

SELECTION-SCREEN BEGIN OF SCREEN 1200 AS SUBSCREEN.
  PARAMETERS : P_CARR2 TYPE SPFLI-CARRID,
               P_CONN2 TYPE SPFLI-CONNID.
SELECTION-SCREEN END OF SCREEN 1200.

SELECTION-SCREEN BEGIN OF SCREEN 1300 AS SUBSCREEN.
  PARAMETERS : P_CARR3 TYPE SFLIGHT-CARRID,
               P_CONN3 TYPE SFLIGHT-CONNID,
               P_DATE3 TYPE SFLIGHT-FLDATE.
SELECTION-SCREEN END OF SCREEN 1300.



SELECTION-SCREEN BEGIN OF TABBED BLOCK MY_TAB FOR 5 LINES.

  SELECTION-SCREEN TAB (10)             " 탭 이름의 넓이
    TABNM1                              " 탭 이름을 가진 변수
    USER-COMMAND UC_TAB1                " 탭 이동을 위한 USER-COMMAND
    DEFAULT SCREEN 1100.                " 탭의 화면을 보여주기 위한 SUB SCREEN

  SELECTION-SCREEN TAB (20)
    TABNM2
    USER-COMMAND UC_TAB2
    DEFAULT SCREEN 1200.

  SELECTION-SCREEN TAB (30)
    TABNM3
    USER-COMMAND UC_TAB3
    DEFAULT SCREEN 1300.

SELECTION-SCREEN END OF BLOCK MY_TAB.


INITIALIZATION.

  TABNM1 = 'AIRLINE'.
  TABNM2 = 'FLIGHT SCHEDULE'.
  TABNM3 = 'FLIGHT'.

* 프로그램 실행 시 2번째 탭인 FLIGHT SCHEDULE 탭이 활성화되어 있도록 조치.
  MY_TAB-ACTIVETAB = 'UC_TAB2'.
  MY_TAB-DYNNR     = 1200.

START-OF-SELECTION.

  CASE MY_TAB-ACTIVETAB.
    WHEN 'UC_TAB1'.
      WRITE /: '현재 AIRPORT 탭이 활성화되었고,',
               'P_CARR1 = ', P_CARR1 , '입니다.'.

    WHEN 'UC_TAB2'.
      WRITE /: '현재 SCHEDULE 탭이 활성화되었고,',
               'P_CARR2 = ', P_CARR2 ,
               'P_CONN2 = ', P_CONN2, '입니다.'.

    WHEN 'UC_TAB3'.
      WRITE /: '현재 FLIGHT 탭이 활성화되었고,',
           'P_CARR3 = ', P_CARR3 ,
           'P_CONN3 = ', P_CONN3 ,
           'P_DATE3 = ', P_DATE3 , '입니다.'.
  ENDCASE.
