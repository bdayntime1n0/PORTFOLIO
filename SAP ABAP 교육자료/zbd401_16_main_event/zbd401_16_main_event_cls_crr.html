" 비행기 종류 만들 때 마다 여기에 보관됨.
CLASS LCL_CARRIER DEFINITION.
  PUBLIC SECTION.
    METHODS CONSTRUCTOR.
    METHODS ON_AIRPLANE_CREATED FOR EVENT AIRPLANE_CREATED OF LCL_AIRPLANE
                                IMPORTING SENDER.

  PRIVATE SECTION.
    DATA MT_AIRPLANE TYPE TABLE OF REF TO LCL_AIRPLANE.
ENDCLASS.

CLASS LCL_CARRIER IMPLEMENTATION.
  METHOD CONSTRUCTOR.
    SET HANDLER ME->ON_AIRPLANE_CREATED FOR ALL INSTANCES.
  ENDMETHOD.

  METHOD ON_AIRPLANE_CREATED.
    " SENDER <-- 새로 만들어진 비행기. SENDER는 LCL_AIRPLANE의 INSTANCE임.
    " 지금 발생한 EVENT는 AIRPLANE_CREATED 인데, 이 EVENT는 LCL_AIRPALNE에서의
    " 생성자 CONSTRUCTOR에 의해 발생된(RAISE) 된 EVENT이기 때문에
    " SENDER 는 TYEP REF TO LCL_AIRPLANE 으로 만들어진 참조변수가
    " CREATE OBJECT 된 순간과 동일함.
    " 결과적으로, 이 METHOD 에서는 SENDER 가 새로 만들어진 비행기라고 봐도 무방.

    APPEND SENDER TO MT_AIRPLANE.
    ENDMETHOD.
ENDCLASS.
