MODULE INIT_ALV_0200 OUTPUT.

  " GO_GRID를 통해서 사용자가 FLDATE 필드를 CLICK 하면서
  " HOT SPOT EVENT가 발생.
  " 이 HOTSPOT 이벤트는 새로운 팝업이 열리며, 사용자가 선택한 라인의 데이터를 기준으로
  " 예약현황을 ALV로 새롭게 보여줌.

  IF GO_GRID_2 IS BOUND.
    " 이미 생성되어 있으면 새로고침.
    PERFORM REFRESH_ALV_GRID USING GO_GRID_2.
  ELSE.
    PERFORM CREATE_CONTAINER_0200.
    PERFORM SET_ALV_LAYOUT_0200.
    PERFORM SET_ALV_FIELDCATALOG_0200.
    PERFORM DISPLAY_ALV_DATA_0200.
  ENDIF.
ENDMODULE.

*&---------------------------------------------------------------------*
MODULE GET_BOOK_0200 OUTPUT.

  REFRESH GT_BOOK.

  SELECT A~BOOKID,
         A~CUSTOMID,
         B~NAME,
         A~CUSTTYPE,
         A~CLASS,
         A~LOCCURAM,
         A~LOCCURKEY,
         A~ORDER_DATE,
         A~CANCELLED,
         A~RESERVED
    FROM SBOOK AS A LEFT OUTER JOIN SCUSTOM AS B
                                 ON B~ID EQ A~CUSTOMID
    WHERE A~CARRID EQ @GS_DISPLAY-CARRID  " GS_DISPLAY 는 HOTSPOT 이벤트에서
      AND A~CONNID EQ @GS_DISPLAY-CONNID  " 클릭한 라인 번호를 기준으로 GT_DISPLAY에서
      AND A~FLDATE EQ @GS_DISPLAY-FLDATE  " 가져와 넣은 후에 CALL SCREEN 0200을 한 것이므로
                                          " 사용자가 넣은 정보가 SCREEN 200에 들어가있음.
    INTO TABLE @GT_BOOK.  " SELECT 에서 가져오는 필드 순서에 따라 그대로 ITAB에 들어감.

" 고객 ID 별 최근 예약한 일자 기준으로 정렬
SORT GT_BOOK BY CUSTOMID
                ORDER_DATE DESCENDING.

ENDMODULE.
