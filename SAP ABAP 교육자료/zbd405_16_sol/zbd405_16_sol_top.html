DATA: GS_FLIGHT TYPE DV_FLIGHTS.

**** SELECTION SCREEN

SELECTION-SCREEN BEGIN OF BLOCK B01 WITH FRAME TITLE TEXT-S01. "SELECITON OPTIONS
  SELECT-OPTIONS : S_CARRID FOR GS_FLIGHT-CARRID MEMORY ID CAR,   "SAP MEMORY에서 DATA를 가져올 때 MEMORY ID라는 문구를 사용.
                   S_CONNID FOR GS_FLIGHT-CONNID,
                   S_FLDATE FOR GS_FLIGHT-FLDATE NO-EXTENSION.  "MULTIPLE SELECTION 제거

SELECTION-SCREEN END OF BLOCK B01.



SELECTION-SCREEN BEGIN OF BLOCK B02 WITH FRAME TITLE TEXT-S02. "ADDITIONAL OPTIONS
  PARAMETERS : P_ALL RADIOBUTTON GROUP R01 USER-COMMAND UC01,
               P_DOM RADIOBUTTON GROUP R01,
               P_INT RADIOBUTTON GROUP R01 DEFAULT 'X'.

  SELECT-OPTIONS : S_CNTYFR FOR GS_FLIGHT-COUNTRYFR NO-EXTENSION
                                                    NO INTERVALS  " HIGH 숨기기
                                                    MODIF ID NOD. " 최대 3글자, SCREEN-GROUP1과 매칭되는 옵션.
  " S_CNTYFR 과 관련있는 모든 화면 OBJECT에
  " GROUP1 필드의 값이 전부 지정된 값으로 관리됨.

SELECTION-SCREEN END OF BLOCK B02.



AT SELECTION-SCREEN OUTPUT.

*  IF P_DOM = 'X' AND S_CNTYFR[] IS INITIAL AND SY-UCOMM NE 'UC01'.

  LOOP AT SCREEN.     " SELECTION-SCREEN 의 모든 OBJECT에 대한 접근.
    " INTERNAL TABLE WITH HEADER LINE.
* 만약 HEADER LINE이 없다면 'LOOP AT SCREEN INTO ~~'로 적혀있어야 함.

    CASE SCREEN-NAME.
      WHEN 'S_CNTYFR-LOW'.            " RANGE TABLE이라 입력필드가 2개이기 때문에 CASE를 사용함.

        IF P_DOM EQ 'X'.
          SCREEN-INPUT = '1'.         " DOMESTIC 인 경우에는 입력을 받겠다.
        ELSE.
          SCREEN-INPUT = '0'.          " S_CNTYFR INPUT에 입력 받지 않겠다.
        ENDIF.

    ENDCASE.

    CASE SCREEN-GROUP1.
      WHEN 'NOD'.                 " DOM을 찍을 때만 입력가능하게 뜸.
        IF P_DOM IS INITIAL.
          SCREEN-ACTIVE = '0'.
        ENDIF.
    ENDCASE.
    MODIFY SCREEN.

  ENDLOOP.



AT SELECTION-SCREEN ON S_CNTYFR.

  IF P_DOM = 'X' AND S_CNTYFR[] IS INITIAL AND SY-UCOMM NE 'UC01'.
    MESSAGE E003(BC405).
  ENDIF.
