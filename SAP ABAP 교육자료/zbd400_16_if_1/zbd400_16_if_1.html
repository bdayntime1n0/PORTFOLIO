REPORT ZBD400_16_IF_1.


PARAMETERS P_TEST TYPE C.

* ELSEIF에 A가 있는 경우 어떻게 될까?

IF P_TEST = 'A'.
  WRITE '입력하신 값은 A 입니다.'.

ELSEIF P_TEST = 'A'.
  WRITE 'IF 문이 다시 수행되었습니다.'.
* 이미 A가 참이기 때문에 아래까지 내려오지 않음.

ELSEIF P_TEST = 'B'.
  WRITE '입력하신 값은 A가 아니고 B 입니다.'.

ELSEIF P_TEST = 'C'.
  WRITE '입력하신 값은 A가 아니고 B가 아니고 C 입니다.'.

ELSE.
  WRITE '입력하신 값은 A가 아니고 B가 아니고 C가 아니고 그 외 입니다.'.

ENDIF.

SKIP 1.

*CASE문

CASE P_TEST.

  WHEN 'A'.
    WRITE '입력하신 값은 A 입니다.'.

  WHEN 'A'.
    WRITE 'CASE 문이 다시 수행되었습니다.'.
* 이미 A가 참이기 때문에 아래까지 내려오지 않음.
  WHEN 'B'.
    WRITE '입력하신 값은 A가 아니고 B 입니다.'.

  WHEN 'C'.
    WRITE '입력하신 값은 A가 아니고 B가 아니고 C 입니다.'.

  WHEN OTHERS.
    WRITE '입력하신 값은 A가 아니고 B가 아니고 C가 아니고 그 외 입니다.'.

ENDCASE.
