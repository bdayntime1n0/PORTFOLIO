REPORT ZBD430_16_SSCR_ON.


SELECTION-SCREEN BEGIN OF BLOCK B01 WITH FRAME TITLE TEXT-S01.

  PARAMETERS P_CARRID TYPE SPFLI-CARRID.
  PARAMETERS P_CONNID TYPE SPFLI-CONNID.

SELECTION-SCREEN END OF BLOCK B01.

AT SELECTION-SCREEN ON P_CARRID.

  IF P_CARRID NE 'LH' AND P_CARRID IS NOT INITIAL.
    MESSAGE E000(OO) WITH 'LH 외에는 입력이 불가능합니다.'.

  ENDIF.

AT SELECTION-SCREEN ON P_CONNID.

  IF P_CONNID IS NOT INITIAL.
    IF P_CONNID NOT BETWEEN '1000' AND '3000'.
      MESSAGE E000(OO) WITH '1000 ~ 3000 사이만 입력이 가능합니다.'.

    ENDIF.
  ENDIF.
