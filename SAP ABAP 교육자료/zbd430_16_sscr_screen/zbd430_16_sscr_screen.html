SELECTION-SCREEN END OF SCREEN 1100.


AT SELECTION-SCREEN ON P_CHK. " P_CHK에 값 변경이 있는 경우 ACTIVE.
  IF P_CHK EQ 'X'.
    CALL SELECTION-SCREEN 1100 STARTING AT 05 05   " 팝업시작위치 열/행

    IF SY-SUBRC EQ 0.
      MESSAGE I000(OO) WITH 'P_CARRID = ' P_CARRID.
    ELSE.
      MESSAGE I000(OO) WITH 'P_CARRID 입력 취소'.
    ENDIF.
  ENDIF.
